Format Email Headers for Jira

May 4, 2018 2 minutes read
notes
python • jira

On a daily basis I find myself forwarding emails to Jira Tickets. When Jira puts the text of the email into an attached comment, the email header is added as text. I like to see that header neatly formatted, so I wrote a Python script to quickly formatted that text before I send it to Jira.

#!/usr/bin/env python3

import pyperclip


def main():
    cliptext = pyperclip.paste()
    try:
        formatted_lines = []

        lines = [line for line in cliptext.split('\n') if line]
        for line in lines:
            field, text = line.split(':', 1)
            formatted = '|| %s | %s |' % (field, text)
            formatted_lines.append(formatted)

        output = '\n'.join(formatted_lines)
        pyperclip.copy(output)
        print(output)

    except Exception as exc:
        print(str(exc))
        print()
        print('Unable to format the following text:')
        print(cliptext)


if __name__ == '__main__':
    main()

To use the script, I copy the email header text I want to format:

From: Jane Doe <jane.doe@example.com>
Sent: Friday, May 4, 2018 8:56 AM
To: Doug Van Horn
Subject: How to make this email header pretty in Jira

Then I run the script from the command line. The script uses the pyperclip library to extract my copied text from the clipboard, format it in a Jira friendly way, and copy the formatted text back to the clipboard.

Then I go back to my email client and I paste right over the already highlighted text.

The code formats the email header into a Jira Table, for easier viewing in the comments:

|| From |  Jane Doe <jane.doe@example.com> |
|| Sent |  Friday, May 4, 2018 8:56 AM |
|| To |  Doug Van Horn |
|| Subject |  How to make this email header pretty in Jira |
